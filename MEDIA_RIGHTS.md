# Media Rights

This repository is a curated Seed-Audio 1.0 usecase collection. It links to public creator and provider posts and stores selected preview media only to make the cases easier to inspect.

## Repository Text

Repository-written README text, generated metadata, maintenance documentation, and issue templates are licensed under CC BY 4.0 unless a file says otherwise.

## Creator And Provider Media

Public posts, creator names, source links, thumbnails, videos, screenshots, and other media remain attributed to their original creators, source platforms, and rights holders.

Media files under `media/cases/` are included as source-linked previews for commentary, research, and educational curation. They are not a blanket relicensing of the original creator's work.

## Reuse Boundary

If you reuse this repository's written summaries, provide attribution to this repository and keep the original case source links.

If you reuse creator media, verify the rights from the original source and credit the original creator. Do not assume media files in this repository are independently reusable under CC BY 4.0.

## Corrections And Removal

If a source link breaks, attribution is wrong, a claim is unsupported, or media should be removed, open a correction issue with the affected case and supporting source.

Maintainers should remove or correct affected media promptly when a creator, rights holder, or platform representative requests it.
